# Sistema de Folha de Pagamento - Nota 100/100 

Projeto acadêmico desenvolvido em Java para a disciplina de Algoritmos e Programação. A solução foi avaliada pela banca universitária e atingiu a pontuação máxima (100 pontos) em todos os critérios de avaliação:

* Aplicabilidade prática da teoria (Geração de valor)
* Profundidade e consistência da solução
* Alinhamento com o desafio e o contexto do case
* Organização e estruturação das informações
* Qualidade da linguagem e da argumentação

## Funcionalidades do Sistema

* Cadastro de Funcionário Padrão: Remuneração base fixa de R$ 2.000,00.
* Cadastro de Funcionário Comissionado: Cálculo baseado no salário fixo somado ao percentual estipulado sobre o total de vendas do mês.
* Cadastro de Funcionário por Produção: Cálculo baseado no salário fixo somado à bonificação pela quantidade de peças produzidas.
* Geração Automática da Folha: Listagem consolidada no terminal exibindo os dados de cada colaborador e realizando o cálculo do salário final dinamicamente.

## Conceitos de POO Aplicados

* Herança: Utilização de classes filhas herdando os atributos e métodos da classe base principal (Funcionario).
* Polimorfismo: Sobrescrita do método de cálculo para permitir que cada categoria de contratação execute sua própria regra de negócio.
