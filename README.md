# Projeto-Sistema_Folha_Pagamento
Projeto de Sistema de Folha de Pagamento em Java desenvolvido para a disciplina de Algoritmos e Programação, recebendo nota máxima (100/100). Aplica conceitos reais de POO (Herança e Polimorfismo) para gerenciar e calcular dinamicamente a remuneração de diferentes categorias de funcionários (Padrão, Comissionado e Produção).
